# Exec Plan Archive

`docs/EXEC_PLAN.md` is reserved for the current active slice.

Use this directory for durable planning notes that are not needed on every
agent turn, such as historical phase trackers, larger design plans, or completed
multi-slice records. Keeping history here prevents the active plan from turning
into a long completion log.

If an older phase tracker is needed, inspect the Git history for
`docs/EXEC_PLAN.md` before Exec Plan V2.
